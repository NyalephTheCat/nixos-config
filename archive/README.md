# Archive Directory

This directory contains old configuration files from the single-host setup, kept for reference.

## Files

- `configuration.nix` - Old single-host NixOS configuration
- `hardware-configuration.nix` - Old hardware configuration (now in hosts/heaven/)
- `home.nix` - Old Home Manager configuration (now in users/nyaleph/)
- `shell.nix` - Old shell configuration

## Note

These files are kept for reference only. The active configuration is now in:
- `hosts/<hostname>/configuration.nix`
- `users/<username>/home.nix`

You can safely delete this directory if you no longer need the reference files.
